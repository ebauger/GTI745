Laboratoire 01
==============

Nous avons fait les modifications F1, F3, D1 et D6

Pour F3
=======

Maintenant, il n’est plus possible de faire un clic droit n’importe où dans l’interface pour changer la couleur d’une boîte. Il faut présentement être sur la boîte pour changer la couleur avec l’aide du clic droit.

Pour D6
=======

Nous avons mis un menu où il est possible d'ajouter des vues de caméra dans le menu de gauche via une liste (JList).
